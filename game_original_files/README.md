# Original Balatro Game Files

## Instructions

**IMPORTANT: You must own a legal copy of Balatro to use this mod.**

1. Purchase Balatro from Steam or another official source
2. Extract the game files to this folder
3. The folder structure should look like:
   ```
   game_original_files/
   ├── resources/
   │   ├── textures/
   │   ├── sounds/
   │   └── ...
   ├── localization/
   │   ├── en-us.lua
   │   └── ...
   └── version.jkr
   ```

## How to Extract

### From Steam (Windows/Linux)
1. Right-click Balatro in Steam → Manage → Browse Local Files
2. Find `Balatro.exe` and extract it using 7-Zip or similar
3. Copy the extracted folders here

### From APK (Android)
1. If you have the Android version, extract the APK
2. Navigate to `assets/` folder
3. Copy contents here
